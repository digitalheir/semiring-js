A semiring is a mathematical construct that abstracts some notion of "plus" and "times" on mathematical objects. These objects can be numbers, but may also be other objects such as strings.

More formally, a semiring is a set RRR equipped with two binary operations ⊕\oplus⊕ and ⊗\otimes⊗ , called addition and multiplication, such that:

  • (R,⊕)(R, \oplus)(R,⊕) is a commutative monoid with identity element 0¯\bar 00¯ :
    • (a⊕b)⊕c=a⊕(b⊕c)(a \oplus b) \oplus c = a \oplus (b \oplus c)(a⊕b)⊕c=a⊕(b⊕c)
    • 0¯⊕a=a⊕0=a\bar0 \oplus a = a \oplus 0 = a0¯⊕a=a⊕0=a
    • a+b=b+aa+b=b+aa+b=b+a
  • (R,⊗)(R, \otimes)(R,⊗) is a monoid with identity element 1¯\bar11¯:
    • (a⊗b)⊗c=a⊗(b⊗c)(a\otimes b)\otimes c=a\otimes (b\otimes c)(a⊗b)⊗c=a⊗(b⊗c)
    • 1¯⊗a=a⊗1¯=a\bar1 \otimes a=a\otimes \bar1 = a1¯⊗a=a⊗1¯=a
  • Multiplication left and right distributes over addition:
    • a⊗(b+c)=(a⊗b)+(a⊗c)a \otimes (b+c)=(a\otimes b)+(a\otimes c)a⊗(b+c)=(a⊗b)+(a⊗c)
    • (a+b)⊗c=(a⊗c)+(b⊗c)(a+b)\otimes c=(a\otimes c)+(b\otimes c)(a+b)⊗c=(a⊗c)+(b⊗c)
  • Multiplication by 0¯\bar 00¯ annihilates R:
    • 0¯⊗a=a⊗0¯=0¯\bar0 \otimes a=a\otimes \bar0 = \bar00¯⊗a=a⊗0¯=0¯
